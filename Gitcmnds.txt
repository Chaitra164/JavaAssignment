Git is a distributed version control system used to manage projects and track changes in files. Here are some common Git commands along with their descriptions:

Basic Commands
git init: Initializes a new Git repository in the current directory.
git clone [url]: Creates a copy of an existing Git repository from a specified URL.
git status: Displays the state of the working directory and the staging area.
git add [file]: Adds a file to the staging area.
git commit -m "[message]": Commits the staged changes with a message.
git push: Uploads local repository content to a remote repository.
git pull: Fetches and merges changes from a remote repository to the local repository.
